# COAL Hotel Booking System

## Description
This project is a simple hotel booking system developed in C++.
It demonstrates COAL concepts like:
- Memory simulation using arrays
- Conditional branching
- Memory update
- Secondary storage (file handling)

## Features
1. Store rooms in memory
2. Book room
3. Prevent double booking
4. Save booking in file

## Navigation Flow
1. Run the program
2. Enter customer name
3. View available rooms
4. Enter room number
5. System books room and saves record
